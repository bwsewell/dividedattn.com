I need to talk about what Claude Code is doing to my brain.  Not in a "robots are taking our jobs" way.  In a "I stayed up until 3am on a Tuesday because I couldn't stop prompting" way.

![A sketch of a MacBook Air with Claude Code running in the terminal](/assets/images/claude-code-terminal.png)

## The First Hit Was Free

I started using Claude Code a few months ago for work stuff.  Scaffolding a project.  Writing some boilerplate.  Debugging a weird CSS issue that would've taken me 45 minutes to Google.  Normal things.  Productive things.

Then one night I thought, "I wonder if it could build me a quick script to organize my Downloads folder."  Twenty minutes later I had a working Python script that sorted files by type and date.  I felt like a genius.  I didn't write a single line of code.

That was the moment everything went sideways.

## The Spiral

Here's what my typical evening looks like now:

1. Sit down after dinner.  Open the laptop to "check one thing."
2. Remember a small project I've been meaning to build.
3. Open Claude Code.  Start prompting.
4. Get a working prototype in 20 minutes.
5. Think, "What if it also did this other thing?"
6. Prompt more.
7. Look at the clock.  It's midnight.
8. Think, "I'm so close though."
9. Look at the clock again.  It's 2:30am.
10. Hate myself.

This has happened at least a dozen times in the past two months.  I am not exaggerating.

## It's Not Like Normal Coding

Here's what's messed up about it.  When I used to code late at night, I'd eventually hit a wall.  A bug I couldn't figure out.  A dependency that wouldn't install.  Some cryptic error message that made me want to throw my laptop out the window.  That friction was a natural stopping point.  It forced me to go to bed.

Claude Code removed the friction.  There's no wall anymore.  You hit an error, you paste it in, and it fixes it.  You want a new feature, you describe it, and it appears.  The feedback loop is so fast that there's no natural break point.  It's like the difference between reading a book (you get tired, you stop) and scrolling TikTok (there's always one more).

The dopamine hit of watching something get built in real time — something *you* described into existence — is genuinely hard to stop.

## The Backlog Problem

The other thing nobody warns you about: Claude Code gave me an ideas problem.  Before, I had plenty of ideas for side projects but never enough time or energy to build them.  That was fine.  The backlog stayed theoretical.  A "someday" list that I'd get to "eventually."

Now I actually *can* build most of them in an evening.  So instead of a backlog of ideas, I have a backlog of half-finished projects.  A receipt printer integration.  A personal dashboard.  A CLI tool for managing my smart home.  A blog (you're reading it).  Each one started at 9pm with the best of intentions and ended at 2am with me staring at a screen wondering where the night went.

The backlog didn't shrink.  It grew.  Because every project spawns three more ideas, and now I know I can build all of them.

## The Guilt Loop

The next day is always the same.  I'm tired.  I'm sluggish.  I'm on my third coffee by noon.  And I feel guilty — not because I wasted time, but because I *was* productive.  It's hard to feel bad about building something.  But my body doesn't care about my git commits.  It cares about the four hours of sleep I got.

This is a weird kind of burnout.  It's not the "I hate my job" kind.  It's the "I love this too much and I can't regulate myself" kind.  It's hobby burnout.  It's the same thing that happens to people who get really into a video game, except instead of leveling up a character, I'm shipping side projects at 1am on a work night.

## What I've Tried

I'm not going to pretend I've solved this.  But I'm trying a few things:

- **Hard cutoff at 11pm.**  Laptop closes.  No exceptions.  I've broken this rule four times already.
- **No Claude Code after dinner on work nights.**  Weekends are fair game.  The week is not.
- **Writing down ideas instead of building them.**  If it's 9pm and I think "what if I built a..." I write it in a notebook and close the laptop.  This works about 60% of the time.
- **Telling my partner to call me out.**  She now says "are you Clauding?" the way people say "are you doomscrolling?" and honestly it's effective.

## I'm Not Quitting

I want to be clear — I'm not going to stop using Claude Code.  It's genuinely one of the best tools I've ever used.  It's made me more productive at work.  It's made side projects actually possible instead of permanently hypothetical.  It's fun.

But it's *too* fun.  And I think a lot of developers are going to hit this same wall as these tools get better.  We talk a lot about AI replacing developers, but nobody's talking about AI making developers unable to stop developing.  The tool is so good that it removed the natural friction that used to protect me from myself.

So yeah.  If you're using Claude Code and you've noticed you're sleeping less, you're not alone.  Set a timer.  Close the laptop.  The project will be there tomorrow.

Your circadian rhythm won't wait.
